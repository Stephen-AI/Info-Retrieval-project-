Discover

This is an Android App that uses three APIs. It uses the FourSquare, Google Places, Google Maps, Picasso, and Firebase APIs. This app allows the user to make a profile with their email and password to log in, and once they are logged in, the app gets their location and sends it to the FourSquare API in an Async Task, and when it gets the JSON file, the data is parsed and put on to a recycler view for every venue. The recycler view is on a fragment, and can easily be replaced by a map that contains markers that show where the venues are located and with an infinity carousel view so that a user can scroll to the next venue while showing them the valuable information on the carousel view. The app has capabilities of going to venue's links, giving the user directions to their selected venue, and seeing the venue photos. This is still in development.
