About Gints

Gints is a software which aims at the dynamic simulation of regulation networks. Its originality comes from the way it is implemented : the main cellular entities are Java objects which interact together. For instance, an RNA polymerase object can “take a gene as argument” and constructs a new RNA object with. A brief description of the main algorithm is available here. Gints is entirely developed with Java. Its XML genes library allows a powerful storage and let people free from the installation of a database management system.

Gints is now under development and some features ask to be improved ( For example, we consider the relation 1 gene gives one protein). Then, if you want to help us to its improvement, do not hesitate !

 If you want a more precise idea, you can also download the PDF presentation with screenshots and/or the manual.
